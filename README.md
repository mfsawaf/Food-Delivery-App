# Food Delivery System (C++)

A console-based food delivery management system inspired by apps like Uber Eats and Talabat.

## Features
- User roles (Customer / Delivery)
- Order creation and tracking
- Driver assignment
- File-based storage
- Binary Search Tree for data management

## Technologies
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Data Structures (BST)

## How to Run
1. Compile the project using any C++ compiler
2. Run main.cpp
